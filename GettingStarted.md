# Introduction #

Phonio provides virtual phones that are used for accepting incoming phones
call; it eases the development of telephony applicaiton.

It uses the twilio.js client library.


# Setting up the numbers #

Twilio numbers are assigned to Phonio users based the 'friendly name';
so if Joe is signed in, and his email is 'joe@example.com', his twilio numbers
should have 'joe@example.com' as the friendly name;
